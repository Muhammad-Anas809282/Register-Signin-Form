SignUp & SignIn Web App Using Local Storage
Project Overview

This project is a simple and responsive web application that allows users to register (sign up) and log in (sign in) using the browser’s local storage. It simulates a basic authentication system without a backend server.

Key Highlights:

Users can register with their name, email, and password.

User credentials are securely stored in the browser’s local storage.

Sign-in functionality verifies credentials against stored data.

After successful login, users are redirected to a dashboard/main page.

Clean and user-friendly UI/UX with responsive design.

Features

User Registration: Capture name, email, and password.

User Login: Authenticate users against stored credentials.

Local Storage Integration: All user data is stored locally in the browser.

Form Reset: Automatically clears input fields after submission.

Dashboard Redirect: Redirects successfully logged-in users to the main page.

Responsive Design: Works smoothly on desktops, tablets, and mobile devices.

Technologies Used

HTML5 – Structure and layout of pages.

CSS3 – Styling and responsive design, including animations.

JavaScript – DOM manipulation, localStorage handling, form validation.

Font Awesome – Icons for UI elements (optional).
